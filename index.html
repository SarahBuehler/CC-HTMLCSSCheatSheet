Common HTML tags

Tag Name Description
<html> html Root element of an HTML document
<head> head Contains machine-readable information (metadata) about the document, like title, scripts and style sheets
<body> body Represents the content of an HTML document
<a> anchor The anchor element, with its hef attribute, creates a hyperlink to web pages, files, email addresses, locations in the same page or a URL
<h1, h2, ..., h6> headings Represents six levels of section headings
<p> paragraph Represents a paragraph, a block of text
<em> emphasis The element marks text that has stress emphasis
<mark> mark Defines marked/highlighted text
<span> span Represents a generic inline container for phrasing content

Centering in CSS

Tag Description
.center {
text-align: center;
} 		Centers inline elements horizontally, wthin a block-level parent element.
.center-me {
margin: 0 auto;
} 		Centers a block-level element by giving it margin-left and margin-right of auto (and it has a set width)
.link {
padding-top: 30px;
padding-bottom: 30px;
} 		Centers a single line vertically, just because there is equal padding above and below it
.center-table p {
vertical-align: middle;
} 		Centers multiple lines vertically with vertical-align
.parent {
display: flex;
flex-direction: column;
justify-content: center;
} 		Centers everything with flex-box (or grid)
